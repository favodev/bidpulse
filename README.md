# BidPulse

## Descripción

BidPulse es una plataforma web de subastas en tiempo real diseñada para listar, pujar y gestionar artículos subastados. Está pensada para:

- Permitir a los usuarios crear y gestionar subastas.
- Ofrecer un sistema de pujas en tiempo real con historial y notificaciones.
- Gestionar perfiles de usuario, valoraciones y favoritos.
- Proveer un panel administrativo para revisar reportes y verificaciones.

El repositorio contiene el frontend en Next.js, integración con Firebase para autenticación, base de datos y almacenamiento, y APIs internas para operaciones del sistema.
